System Operational Manuals

The "Brain" - Booking Workflow
1. How to Handle a New Request:

- Navigate to Booking > Add Booking.
- If a call arrives, select the Client and log their request as Full Day, Local, or Airport.
- Immediately select a Vehicle from the dropdown to verify availability.
- Check the "Notify Driver" box—this sends an automatic SMS to the Driver and Client with details.
The "Hands" - Duty Slip Rules
2. Processing Completed Trips:

- Review the Missing Report on the Home Dashboard daily.
- Call drivers who have unsubmitted Duty Slips.
- Once received physically, go to Duty Slip > Enter Duty Slip.
- Enter exact Closing KM and Times—these numbers are required to power the automatic Billing engine.
The "Wallet" - Automated Invoicing
3. Generating Client Bills:

- Go to Invoice > Generate Invoice.
- Every successfully completed Duty Slip sits in a pending-bill queue.
- Select the Duty Slip; the system automatically matches the assigned Rate Card to calculate the amount.
- Process generation and designate if the bill must be sent via WhatsApp, Email, or the GeM Portal.
Nudge System - Payments
4. Ensuring Revenue Collection:

- Visit Payment > All Payments to view Overdue Invoices.
- Use the Nudge Client buttons to prompt automatic follow-ups configured to company-specific delays (e.g. Every 7 days).
- When Bank updates reflect payment, post it immediately to mark the invoice as Paid in Reports.
